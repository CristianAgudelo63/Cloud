    • ¿Quiénes Somos?

    Somos un servicio nube de código abierto, en el cual se puede almacenar archivos. Se puede almacenar archivos de tipo video, documentos e imágenes, ¡Desde cualquier dispositivo conectado a la red local!

    Ventajas de utilizar FESACLOUD

  • Es una nube de tipo Platform as a Service (PaaS)
  • Es la nube digital para instituciones educativas
  • Es una plataforma multiusuario y fácil de usar

    • Tecnologías Utilizadas

    Las Tecnologías utiliza en FESACLOUD, son las tecnologías del desarrollo web moderno con HTML5, CSS3, JavaScript ES6 (ECMAScript 6), React JS, FireBase; gestión de paquetes con npm; gestores de controles de versiones con Git y GitHub; finalmente un contenedor con Docker de toda la aplicación.

    Descargar

    ¡Descarga nuestra nube para tu Institución Educativa!. Ingresa a GitHub o clona nuestra plataforma de código abierto desde git en Windows, Linux o MacOS:git clone https://github.com/CristianAgudelo63/Cloud.git

    Preguntas Frecuentes

    ¿Tiene algún costo utilizar la plataforma?

    Utilizar la plataforma es totalmente gratuito. Sin embargo, se ofrece planes con beneficios, dependiendo del plan con el que cuentes, tu capacidad de almacenamiento aumenta.

    ¿Cuales son los métodos de pago?

    Nuestros métodos de pago son por efectivo, por cuenta de ahorro Bancolombia y Davivienda.

    ¿Para quién va orientado FESACLOUD?

    Está orientado hacia Instituciones Educativas que desean tener los datos de sus estudiantes y maestros protegidos y totalmente seguros.

    ¿Dónde queda almacenada la información?

    En un servidor local instalado en la institución destinada. Actualmente solo se pude acceder a la plataforma de forma local en las instituciones.
